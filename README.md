# 匿名ウェブプロキシ

シンプルな匿名プロキシサーバー。
クライアントと外部サーバーの間に入り、IPアドレスや特定ヘッダーを公開せずに通信を中継します。

---

## 概要

このプロジェクトは以下を目的としています：

* クライアントIPの秘匿
* 不要な転送ヘッダーの削除
* 軽量で扱いやすいプロキシの提供

HTTPおよびHTTPSトラフィックに対応しています。

---

## 特徴

* HTTP / HTTPS 対応
* Via ヘッダーの除去
* ログ制御可能
* 環境変数による設定

---

## ホスト方法

Renderや自宅サーバーなどでホストしてください。
nodejsのインストールが必須です。※nodejs24以上を推奨します。

```bash
npm install
```
を実行して依存関係をインストールしてください。

## Running the Server

```bash
npm start
```

デフォルト設定：

* Host: 0.0.0.0
* Port: 3000

アクセス例：

```
http://localhost:3000
```

---

## 必須設定

環境変数で設定してください。

| Name        | Description |
| ----------- | ----------- |
| JWT_SECRET  | JWTキー        |
| LOGIN_USER  | ログイン時のユーザー名 |
| LOGIN_PASS  | パスワード |

例：

```bash
JWT_SECRET=abcdefghijklmsopqrstuvwxyz
LOGIN_USER=username
LOGIN_PASS=1234
```
